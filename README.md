Sentiment-Anaysis
=================

Sentiment Analysis of twitter live stream data using AFINN text. This is done as a part of the Introduction to Data Science Coursera Course.

You must have python 2.7 installed.
Output.txt contains snapshot of twitter livestream data captured using the openly available Twitter API.
AFINN-111.txt contains the AFINN words and their associated sentiment scores.
tweet_sentiment.py computes the sentiment score of the tweets in output.txt using the AFINN words.
term_sentiment.py computes the sentiment scores of non-AFINN words from the sentiment score of the tweet.
